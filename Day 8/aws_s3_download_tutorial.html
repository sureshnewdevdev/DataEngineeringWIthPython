AWS S3 File Download Using Python

Step-by-step classroom-style guide to download files from an Amazon S3 bucket using Python and boto3.

Objective

This tutorial explains how to access an AWS S3 bucket and download files using a Python program.

The program can download:

  • All files from an S3 bucket
  • Files from a specific S3 folder/prefix
  • Files while preserving folder structure locally

Step 1: Check Python Installation

Open Command Prompt and run:

python --version

or:

py --version

If Python is not installed, install Python first.

Step 2: Install AWS CLI

Check whether AWS CLI is installed:

aws --version

If it is not installed, install AWS CLI Version 2. After installation, run the same command again.

Step 3: Configure AWS Credentials

Run this command:

aws configure

Enter the following values:

AWS Access Key ID [None]: your-access-key-id
AWS Secret Access Key [None]: your-secret-access-key
Default region name [None]: ap-south-1
Default output format [None]: json
Important: Never share your AWS Secret Access Key with anyone. Do not paste it in GitHub, WhatsApp, or screenshots.

Step 4: Required IAM Permissions

The IAM user should have permission to list and download files from the bucket.

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "s3:ListBucket"
      ],
      "Resource": "arn:aws:s3:::YOUR_BUCKET_NAME"
    },
    {
      "Effect": "Allow",
      "Action": [
        "s3:GetObject"
      ],
      "Resource": "arn:aws:s3:::YOUR_BUCKET_NAME/*"
    }
  ]
}

Replace YOUR_BUCKET_NAME with your real bucket name.

Step 5: Test S3 Access

List all buckets:

aws s3 ls

List files inside a specific bucket:

aws s3 ls s3://YOUR_BUCKET_NAME

Step 6: Install boto3

boto3 is the AWS SDK for Python.

pip install boto3

or:

py -m pip install boto3

Step 7: Python Program

Create a file named download_s3_files.py and paste this code:

import boto3
import os
from botocore.exceptions import ClientError, NoCredentialsError


def download_files_from_s3(bucket_name, s3_prefix, local_download_folder):
    """
    Download files from an AWS S3 bucket.

    bucket_name: Name of the S3 bucket.
    s3_prefix: Folder path inside bucket. Use empty string "" to download all files.
    local_download_folder: Local folder where files will be downloaded.
    """

    try:
        s3_client = boto3.client("s3")
        paginator = s3_client.get_paginator("list_objects_v2")

        pages = paginator.paginate(
            Bucket=bucket_name,
            Prefix=s3_prefix
        )

        file_count = 0

        for page in pages:
            if "Contents" not in page:
                print("No files found in the given S3 path.")
                return

            for obj in page["Contents"]:
                s3_key = obj["Key"]

                if s3_key.endswith("/"):
                    continue

                relative_path = s3_key[len(s3_prefix):] if s3_prefix else s3_key
                local_file_path = os.path.join(local_download_folder, relative_path)

                local_dir = os.path.dirname(local_file_path)
                if local_dir:
                    os.makedirs(local_dir, exist_ok=True)

                print(f"Downloading: s3://{bucket_name}/{s3_key}")
                print(f"Saving to   : {local_file_path}")

                s3_client.download_file(
                    bucket_name,
                    s3_key,
                    local_file_path
                )

                file_count += 1

        print("--------------------------------")
        print("Download completed successfully.")
        print(f"Total files downloaded: {file_count}")

    except NoCredentialsError:
        print("AWS credentials not found.")
        print("Please run: aws configure")

    except ClientError as error:
        print("AWS error occurred:")
        print(error)

    except Exception as error:
        print("Unexpected error occurred:")
        print(error)


if __name__ == "__main__":
    BUCKET_NAME = "YOUR_BUCKET_NAME"
    S3_PREFIX = ""
    LOCAL_DOWNLOAD_FOLDER = "s3_downloads"

    download_files_from_s3(
        BUCKET_NAME,
        S3_PREFIX,
        LOCAL_DOWNLOAD_FOLDER
    )

Step 8: Update Bucket Name

Change this line:

BUCKET_NAME = "YOUR_BUCKET_NAME"

Example:

BUCKET_NAME = "my-company-bucket"

Step 9: Download All Files

To download all files from the bucket, keep:

S3_PREFIX = ""

Step 10: Download Files from One Folder

If your S3 path is:

s3://my-company-bucket/documents/

Use:

S3_PREFIX = "documents/"

Step 11: Run the Program

Run:

python download_s3_files.py

or:

py download_s3_files.py

Downloaded files will be saved inside:

s3_downloads

Common Errors and Fixes

Error Meaning Fix
Unable to locate credentials AWS credentials are not configured Run aws configure
Access Denied IAM user does not have permission Add s3:ListBucket and s3:GetObject
No files found Bucket or prefix may be wrong Check bucket name and S3_PREFIX

Memory Points

  • S3 bucket means main storage container.
  • S3 prefix means folder path inside bucket.
  • S3 key means full file path inside bucket.
  • download_file() downloads one file.
  • list_objects_v2() lists files from S3.
